State Schema

Define a shared LangGraph state object:

class VideoAgentState(TypedDict):
text: str
acting_prompt: str
reference_voice: str
face_image: str
audio_path: str
raw_video_path: str
final_video_path: str

Each node in the graph must read from and update this state.

Construct a LangGraph pipeline with nodes:

render

→ voice_clone

→ END
